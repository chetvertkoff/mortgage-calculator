
# mortgage-calculator

## Project layers

App - Vue app layer

Domain - domain, use case layer

Store - data layer (Vue apollo usage)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
### Unit(Jest) testing 
```
yarn test
```
### E2E(Cypress) testing
```
yarn test:e2e
```
